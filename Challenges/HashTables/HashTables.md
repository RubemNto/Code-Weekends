HashTable
0 -> 1 2 3

EXAMPLE
HashTable
0 -> 1
1 -> 2
2 -> 3
ChangeItemValue(HashTable, 0, 2)

        0 -> 2

        2 -> 3

EXAMPLE
HashTable

"2" -> 10

        "1" -> 2
